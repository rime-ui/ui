## API Reference

### Props

| Prop        | Type                                                                 | Default     |
| ----------- | -------------------------------------------------------------------- | ----------- |
| `variant`   | `'primary' \| 'secondary' \| 'ghost' \| 'dangerous' \| 'gold'`       | `'primary'` |
| `size`      | `'small' \| 'medium' \| 'large'`                                     | `'medium'`  |
| `outline`   | `boolean`                                                            | `false`     |
| `className` | `string`                                                             | —           |

The `Button` component also accepts all standard `button` HTML attributes.
