## Custom Max Value

By default, the max value is 100. You can change this to represent different scales (e.g., steps in a form).

<ComponentPreview
  preview={
    <div className="w-full max-w-md p-4 space-y-2">
       <span className="text-sm text-gray-500">Step 2 of 5</span>
       <Progress value={2} max={5} variant="gold" />
    </div>
  }
  code={`<Progress value={2} max={5} variant="gold" />`}
/>

## API Reference

### Props

| Prop           | Type                                                           | Default     | Description                                |
| -------------- | -------------------------------------------------------------- | ----------- | ------------------------------------------ |
| `value`        | `number`                                                       | `0`         | The current progress value.                |
| `max`          | `number`                                                       | `100`       | The maximum progress value.                |
| `variant`      | `'primary' \| 'secondary' \| 'ghost' \| 'dangerous' \| 'gold'` | `'primary'` | The visual style of the progress bar.      |
| `size`         | `'small' \| 'medium' \| 'large'`                               | `'medium'`  | The height/thickness of the bar.           |
| `striped`      | `boolean`                                                      | `false`     | Shows an animated striped pattern.         |
| `className`    | `string`                                                       | —           | Class name for the outer track container.  |
| `barClassName` | `string`                                                       | —           | Class name for the inner progress indicator.|

The `Progress` component also accepts all standard `div` HTML attributes.
