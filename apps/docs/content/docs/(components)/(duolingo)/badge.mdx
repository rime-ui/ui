| `variant`   | `'primary' \| 'secondary' \| 'ghost' \| 'dangerous' \| 'gold'` | `'primary'` |
| `children`  | `string`                                                       | —           |
| `className` | `string`                                                       | —           |

The `Badge` component renders a `div` element.
