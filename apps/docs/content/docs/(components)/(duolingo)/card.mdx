# Card

A flexible and composable card component used to group related content and actions. It follows the compound component pattern.

## Usage

<CodeBlock
  language="tsx"
  code={`import { Card, Button } from "@rime-ui/ui"

export function Example() {
  return (
    <Card.Root>
      <Card.Header>
        <Card.Title>Card Title</Card.Title>
      </Card.Header>
      <Card.Body>
        Content goes here...
      </Card.Body>
      <Card.Footer>
        <Button>Action</Button>
      </Card.Footer>
    </Card.Root>
  )
}`}
/>

## API Reference

The Card component relies on a compound component pattern.

### Components

| Component     | Description                                               |
| ------------- | --------------------------------------------------------- |
| `Card.Root`   | The main container for the card.                          |
| `Card.Header` | Container for the card's title and description.           |
| `Card.Title`  | The main heading of the card.                             |
| `Card.Body`   | The main content area of the card.                        |
| `Card.Footer` | Container for actions (buttons) at the bottom.            |

### Props

All Card sub-components accept standard HTML attributes (e.g., `className`, `children`, `dir`).
