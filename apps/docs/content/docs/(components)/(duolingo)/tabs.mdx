## API Reference

The Tabs component relies on a compound component pattern.

### Components

| Component      | Description                                               |
| -------------- | --------------------------------------------------------- |
| `Tabs.Root`    | The main container that manages state.                    |
| `Tabs.List`    | The container for the tab triggers.                       |
| `Tabs.Tab`     | The button that activates a specific content panel.       |
| `Tabs.Content` | The content panel associated with a tab.                  |

### Props

#### Tabs.Root

| Prop           | Type        | Default | Description                                  |
| -------------- | ----------- | ------- | -------------------------------------------- |
| `defaultValue` | `string`    | —       | The value of the tab that should be active initially. |
| `className`    | `string`    | —       | Additional classes for the container.        |

#### Tabs.Tab

| Prop           | Type        | Default | Description                                  |
| -------------- | ----------- | ------- | -------------------------------------------- |
| `value`        | `string`    | —       | A unique value to identify this tab.         |
| `className`    | `string`    | —       | Additional classes for the tab trigger.      |

#### Tabs.Content

| Prop           | Type        | Default | Description                                  |
| -------------- | ----------- | ------- | -------------------------------------------- |
| `value`        | `string`    | —       | Must match the value of the associated tab.  |
| `className`    | `string`    | —       | Additional classes for the content wrapper.  |
